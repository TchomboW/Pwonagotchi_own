# Pwonagotchi (Optimized High-Performance Edition)

This is a high-density, asynchronous optimization of the original pwnagotchi project.

## 🚀 Deployment Instruction
1. Flash Raspberry Pi OS Lite (64-bit) to your SD card via Pi Imager.
2. Run: `curl -sSL https://raw.githubusercontent.com/TchomboW/Pwonagotchi_own/main/setup.sh | bash`
